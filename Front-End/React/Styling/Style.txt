Styling react componenet
========================

1- CSS STYLESHEETS
    - Create css files and import them.
    - In react when applying applying a style in JSX, we use property called className.
        - Ex: <h1 className='primary'>Title</h1>
    - We can also apply styles conditionally. 
        - Ex: having a condition: let myStyle = "". In jsx line className = {myStyle}. 
    - If we want to specify Multiple classes, we can use temperal literals in the property className. 

2- CSS MODULES.
    - CSS module is a feature available with react scripts version 2 or higher.
    - There is a file naming convention to be used for CSS modules.
        - File name must be suffixed with '.module.css'

3- Use tailwind
